# Learn NumPy in One Day

## 🚀 Learning Roadmap

To master the material effectively, follow this file order:

1. **`basics.py`** — Installation, importing, and creating your first arrays.
2. **`dimensions.py`** — Working with 1D, 2D, and 3D arrays and checking `.shape`.
3. **`slicing.py`** — Mastering index-based slicing for rows, columns, and quadrants.
4. **`arithmetic.py`** — Scalar operations, vectorized math functions, and constants.
5. **`broadcasting.py`** — Operations on arrays with different shapes.
6. **`aggregation.py`** — Data summary: sums, means, and axis-based analysis.
7. **`filtering.py`** — Boolean indexing and the `np.where()` function.
8. **`random_data.py`** — Generating random numbers, shuffling, and random choices.

## 📂 Project Structure

* **`src/`** — Source code for all NumPy lessons.
* **`README.md`** — Project documentation and roadmap.
* **`.venv/`** — Python virtual environment.
